# Portfolio Starterkit

This is a simple portfolio starter kit for you to download and adapt to your needs.

## Table of Contents

* [Instructions](#instructions)
* [Dependencies](#dependencies)
* [Acknowledgements](#acknowledgements)

## Instructions

* Save the files to your laptop:
    * You can either clone the repository
    * Or download the zip file from this Github account
* Open index.html in the browser using a link that looks similar to: /Users/username/projects/Portfolio_StarterKit/index.html
    * NOTE:You will need to adjust the exact path to match your folder structure
* Open the downloaded folder in Atom (or your preferred text editor)
* Adjust the styles and content to make it your own
* Refresh index.html in the browser to see the changes


## Dependencies

This site was built with [JQuery](https://jquery.com/) and [Bootstrap](https://getbootstrap.com/).
